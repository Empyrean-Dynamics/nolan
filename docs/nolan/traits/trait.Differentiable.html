fn variable(value: f64, param_idx: usize) -> Self

Initialize a variable with unit gradient at param_idx.

For scalar types (f64), returns the value and ignores the index. For Jet types, seeds the gradient vector with 1.0 at param_idx.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
